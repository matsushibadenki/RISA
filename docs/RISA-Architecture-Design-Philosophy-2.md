重要なのは、「第三の知能方式」と言っても、**ANNやSNNを単に置き換えるもの**として考えるよりも、**知能とは何かを一段抽象化して考え直す**という立場になることです。

現在のAIを非常に大雑把に分類すると、

```text
第一世代
=============
ルールベースAI
(IF-THEN)

知識 = 人間が書く
```

↓

```text
第二世代
=============
ニューラルネットワーク

知識 = 重み
```

↓

```text
第三世代？（仮）
==================
構造自己組織化AI

知識 = 関係構造
```

という流れになります。

ここで一番大きい違いは、「学習対象」です。

| 世代     | 学習するもの |
| ------ | ------ |
| ルールベース | ルール    |
| ANN    | 重み     |
| SNN    | 重み＋時間  |
| 第三方式   | 構造そのもの |

つまり、

```text
A → B
```

という一本の接続を強くするのではなく、

```text
A
 \
  \
   C
  /
 /
B
```

という**新しい構造そのもの**を生成します。
